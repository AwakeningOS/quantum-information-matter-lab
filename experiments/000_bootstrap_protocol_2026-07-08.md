# Bootstrap Protocol: Contextual Information Matter Lab

Date: 2026-07-08

## Question

Can the old reproducibility discipline and generic sandbox components be migrated into a clean repository for the next phase: contextual information matter?

## Layer

```text
META / repository bootstrap
CLASSICAL_COMPONENT baseline
CONTEXTUAL_COMPONENT starter scaffold
```

## What is migrated

```text
1. operating discipline: no code + no raw log = no result
2. classical-effective information microreactor sandbox baseline
3. status-gate and raw-log checker structure
4. contextual component skeleton for future experiments
```

## What is not migrated

```text
old raw logs
old paper-specific result claims
unverified historical notes
life/cell/metabolism language as claims
hardware result claims
```

## Success criteria

```text
README.md exists and states current non-claims
SKILLS.md exists and defines operating rules
results/STATUS.md exists and has no experimental claims yet
scripts/check_raw_logs.py runs bootstrap generators
new contextual scaffold writes raw JSON
```

## Forbidden claims

This bootstrap does not claim:

```text
life-like behavior
cells
metabolism
self-repair
self-regulation
quantum advantage
physical matter synthesis
```
